# Basic_CRUD_practice
스프링부트 학습용 기초 CRUD 프로젝트 입니다.

- 개발 기간
    - 22/12/22 ~ 22/12/24 (업데이트중)
    
- 개발 환경
    - Java 17
    - Spring Boot : v3.0.0
    - JPA
    - Thymeleaf
    - H2 Database
    
- 주요 기능
